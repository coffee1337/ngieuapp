# ✨ О проекте

NGIE University App — это pet-project, представляющий собой современное приложение для университета с поддержкой нескольких платформ.

Проект был создан для практики:
- Flutter-разработки;
- построения UI;
- организации архитектуры приложения;
- работы с кроссплатформенной разработкой;
- адаптивного интерфейса.

Основной акцент сделан на:
- современный интерфейс;
- компонентный подход;
- структуру приложения;
- работу с Flutter ecosystem.

# ⚙️ Установка и запуск

## Клонирование репозитория

```bash
git clone https://github.com/coffee1337/ngieuapp.git
cd ngieuapp
```

---

## Установка зависимостей

```bash
flutter pub get
```

---

## Запуск приложения

```bash
flutter run
```

---

## Сборка production-версии

### Android

```bash
flutter build apk
```

### Web

```bash
flutter build web
```
